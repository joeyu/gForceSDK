# File Structure

```
root directory
  |
  +- binding - bindings to different programming languages
  |  |
  |  +- (Unity3D, java, python, javasscript, etc)
  |
  +- doc - documents
  |
  +- example - examples
  |
  +- external - 3rd-parting software used by this software
  |
  +- inc - contains header files of APIs
  |
  +- lib - dependent libraries that we released
  |  |
  |  +- (linux, windows, ios, osx, android, etc)
  |
  +- script - IDE workspace/project files and build scripts
  |  |
  |  +- (vs2013, g++, etc)
  |
  +- src - main source files
  |  |
  |  +- ble - BLE library
  |  |  |
  |  |  +- npi - TI NPI library
  |  |     |
  |  |     +- npiutil - utility of TI NPI
  |  |
  |  +- common - common shared files
  |
  +- test - tests
```
